# hello-pi

Collection of project for Raspberry PI 3 + "Basic Starter Learning Kit For Raspberry PI"

# Requirements

* Ubuntu
* Raspberry 3
* Basic Starter Learning Kit For Raspberry PI
    * Prototyping board
    * LED
    * Mini-servo SG90
